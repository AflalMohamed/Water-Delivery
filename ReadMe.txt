Water Delivery Qatar - Project Readme

Functions in this project:

1. Admin login page (login.php)
2. Admin can add, edit, delete water types (name, weight, price, image)
3. Upload and manage images for each water type
4. View all water types on the public side (water.php)
5. About Us page with details and image (about.php)
6. Uses Bootstrap 5 for responsive design
7. MySQL database connection (includes/db.php)

Folder structure:
- includes/ : database connection file
- uploads/ : uploaded images
- index.php : home page
- water.php : display water types
- about.php : about page
- login.php : admin login
